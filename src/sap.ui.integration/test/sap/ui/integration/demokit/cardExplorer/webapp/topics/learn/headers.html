Integration Card Headers

The card header displays essential information about the card, such as its title, subtitle, and status of the card. It groups a set of card attributes and presents them at the top of the card.

This section describes the different header types that can be used in the card header area ("sap.card"/header in the manifest).

Note: "sap.card"/header is mandatory. Providing it together with title is essential for proper screen reader support and other functionalities. If needed, it can be visually hidden using its visible property.

Header Types

Two types of card headers are available:

  • Default
  • Numeric

Additionally, both header types can be extended with the Info Section feature: Extended header with info section
